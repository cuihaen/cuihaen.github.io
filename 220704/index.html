html 기본 태그 모음(1)
html
2022.07.04.

1. <h1> ~ <h6>

→ 헤드라인 태그(제목 ‣ 목차 outline)

→ 웹 접근성을 높이기 위해서는 ‘목차(outline)‘가 반드시 필요.

→ 하나의 html페이지에는 꼭 한 개의 <h1>태그가 존재하게 된다. (일반적으로 사이트 좌측 최상단 로고에 사용)

→ <h1> ~ <h6>태그를 통해 단계를 표현할 때는 중간 단계를 절대 생략하지 않는다.(하위단계 사용)

2. <p>

→ 문단 태그. 기본css로 위 아래에 여백을 가지고 있음.

3. <div>

→ 빈 박스(상자) ‣ 레이아웃을 구성하는 주 태그.

→ 기본CSS가 전혀 없음. ‣ 여백(margin)도 가지고 있지 않음.

특정 키워드 / 단어 / 문장의 시작적인 표현이나 강조를 하기 위한 요소.

<span> → css를 사용해서 시작적 표현(주관적인 표현). <strong> → font-weight:bold;(강한 강조, 리더기가 읽을 때 별도로 큰 볼륨으로 강조해서 읽어주게 됨) <em> → font-style:italic; (약한 강조, 리더기가 읽을 때 별도의 조금 큰 볼륨으로 강조해서 읽어주게 됨)

별도로 화면상의 text를 줄바꿈 할 때는
태그를 활용해 표현. → 일반적인 enter키의 효과.

띄어쓰기의 경우 &ndsp;를 통해 표현하지만 되도록이면 사용하지 않음.

그 외에 문자표를 표현하고 싶을 때는 유니코드를 통해 입력하는 것이 일반적.

(ex ‣ &copy; = ⓒ / &lt; = < / &gt; = > / &amp; = &)
