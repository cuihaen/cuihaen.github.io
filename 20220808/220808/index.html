🟡 테이블 처리

  • 행 단위로 잡는 방법 (odd:홀수, even:짝수) 
    table tr:nth-child(odd){background: #eee}
table tr:nth-child(even){background: rgb(245, 209, 209)}
table tr:first-child{background: orange}
    table tr:nth-of-type(odd){background: red}
table tr:nth-of-type(even){background: gray}
  • 열 단위로 잡는 방법 = nth-child와 nth-of-type 모두 사용 가능!
    table td:nth-child(4n){background: grey}
//*n은 배수의 역할을 함.
    table td:nth-child(4n){background: grey;}
table td:nth-child(4n+1){background: darkcyan}
table td:nth-child(4n+2){background: darkolivegreen}
table td:nth-child(4n+3){background: darkgoldenrod}
